# Quotely - Random Quote Generator

A modern, responsive, and elegant **Random Quote Generator** built with **React, TypeScript, Vite, and Tailwind CSS**. Generate inspirational quotes instantly with a beautiful user interface, dark/light mode support, favorites, sharing options, and smooth animations.

## ✨ Features

### 📌 Core Functionality

* Displays a random quote every time the application loads.
* **New Quote** button generates a different quote on every click.
* Displays both the **quote text** and **author** clearly.
* Prevents consecutive duplicate quotes.
* Quote counter showing current quote and total generated.

### 🎨 User Experience

* 🌞 Light Theme
* 🌙 Dark Theme
* Theme preference stored using Local Storage.
* Smooth fade and transition animations.
* Fully responsive design for Desktop, Tablet, and Mobile.

### ❤️ Additional Features

* Copy quote to clipboard.
* Share quotes to:

  * X (Twitter)
  * WhatsApp
  * LinkedIn
* Save favorite quotes.
* Favorite quotes stored using Local Storage.
* Loading animation while generating quotes.
* Toast notifications for user actions.

### ♿ Accessibility

* Keyboard accessible.
* Proper ARIA labels.
* High contrast colors.
* Visible keyboard focus indicators.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YourUsername/CodeAlpha_RandomQuoteGenerator.git
```

Navigate into the project:

```bash
cd CodeAlpha_RandomQuoteGenerator
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
